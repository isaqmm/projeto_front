# Palhoça Hero

## Sobre o Projeto

O Palhoça Hero é um sistema desenvolvido com o objetivo de incentivar os cidadãos a registrar problemas urbanos da cidade através de um mapa interativo.

Os usuários podem registrar ocorrências, ganhar pontos de experiência (XP), subir de patente e desbloquear conquistas.

## Como Executar

1. Baixe ou clone o projeto.
2. Abra a pasta do projeto.
3. Execute o arquivo `index.html` em um navegador.

Não é necessário instalar banco de dados ou servidor.

## Funcionamento

O usuário cria uma conta e realiza login no sistema. Ao clicar em um ponto do mapa é possível registrar uma ocorrência. Cada ocorrência gera XP para o usuário, permitindo subir de patente e desbloquear conquistas.
